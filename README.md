# Personal Finance Tracker

## Project Description
The Personal Finance Tracker web application helps users track their income and expenses, set budget targets, and gain AI-powered financial insights to make smarter financial decisions. Designed for college students and young professionals, it simplifies money management and encourages better financial habits.

## Core Features
- Income & Expense Tracker
- Budget Targets & Alerts
- AI-driven Insights
- Visual Analytics with charts and graphs

## Technology Stack
- **Frontend:** React
- **Backend:** Node.js / Express
- **Database:** MongoDB
- **AI Integration:** OpenAI API
- **Deployment:** Docker, Google Cloud
- **CI/CD:** GitHub Actions

## Getting Started
1. Clone the repository:
   git clone https://github.com/cschrock1/personal-finance-tracker.git
2. Install dependencies:
   npm install
3. Set up environment variables:
   cp .env.example .env
4. Start the app:
   npm run dev

## AI Integration
- OpenAI API is used to provide personalized financial insights.
- Example: Users receive spending recommendations based on their transaction history.
- AI was also used during development for generating boilerplate code and debugging.
